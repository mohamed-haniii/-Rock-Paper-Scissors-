# Rock Paper Scissors

A console-based Rock Paper Scissors game built with **C++**. The game lets the player compete against the computer over multiple rounds, tracks wins and draws, and prints the final winner.

## Features

- Player vs computer gameplay.
- Random computer choice generation.
- Round-by-round winner calculation.
- Final game result summary.
- Replay loop after game completion.
- Clean use of enums, structs, and functions.

## Tech Stack

- C++
- Console application
- Standard Library

## Skills Demonstrated

- Procedural programming in C++.
- Using enums to model fixed choices and winners.
- Using structs to organize round and game results.
- Random number generation with `rand()` and `srand()`.
- Input validation and control flow.
- Function-based code organization.

## How To Run

Compile and run the C++ file:

```bash
g++ Untitled-1.cpp -o rock-paper-scissors
./rock-paper-scissors
```

On Windows:

```powershell
g++ Untitled-1.cpp -o rock-paper-scissors.exe
.\rock-paper-scissors.exe
```
